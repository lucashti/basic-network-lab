# 🌐 Basic Network Lab

Simple LAN simulation created using Cisco Packet Tracer.

---

## 📌 About This Project

This lab was created to practice basic network configuration and understand how devices communicate inside a Local Area Network (LAN).

---

## 🖧 Network Topology

- 1 Router
- 1 Switch
- 3 PCs
- Star topology

All devices are connected through a central switch.

---

## 📡 IP Addressing Scheme

Network: 192.168.0.0/24

- Router (Gateway): 192.168.0.1
- PC1: 192.168.0.2
- PC2: 192.168.0.3
- PC3: 192.168.0.4

Subnet Mask: 255.255.255.0  
Default Gateway: 192.168.0.1  

---

## 🔧 Configuration Steps

1. Connect devices using copper straight-through cables.
2. Configure router interface with IP address.
3. Assign static IP addresses to each PC.
4. Configure default gateway.
5. Test connectivity using `ping` command.

---

## 🧠 Concepts Practiced

- LAN configuration
- IPv4 addressing
- Subnetting basics
- Default gateway
- Router and switch roles
- Basic troubleshooting with ping

---

## 🎯 Objective

Strengthen understanding of network infrastructure, IP configuration and connectivity testing.
